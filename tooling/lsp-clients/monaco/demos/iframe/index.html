Iframe Demo

Drop-in embed via a script tag — no npm or bundler required.

1. Include the script

Copy dist-iframe/ to your static assets, then add a single script tag: 

<script src="kson-editor.js"></script>

2. Create an editor

const editor = await KsonEditor.create(
    document.getElementById('editor'),
    {
        value: '{ name: "hello" }',
        schema: {
            fileExtension: 'kson',
            schemaContent: mySchema,
        },
        onChange(value) {
            console.log('changed:', value);
        },
    },
);

3. Read and write content

editor.getValue();
editor.setValue('{\n  name: "updated"\n}');

4. Clean up

editor.dispose();

Options

{
  value?: string,
  uri?: string,
  schema?: {
    fileExtension: string,
    schemaContent: string,
  },
  editorOptions?: Record<string, unknown>,
  onChange?: (value: string) => void,
  baseUrl?: string,
}
