# Iframe Demo

Drop-in integration: include `kson-editor.js` from `dist-iframe/` via a
`<script>` tag — no bundler required.
See [`../readme.md`](../readme.md) for the full set of integration examples.

```bash
./gradlew tooling:lsp-clients:npm_run_demoIframe
```

Serves on [http://localhost:5175](http://localhost:5175).
